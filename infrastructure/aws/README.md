# AWS Automation

Stand up the physical layer of this application on AWS.

## :rocket: Quick Start

1. Install Python packages

  ```shell
  pyton3 -m pip install -r requirements.txt
  ```

2. Deploy resources on AWS

  ```shell
  python3 -m setup
  ```
